Pomodoro Timer

Organization

  • SpartaHack VII Hackathon

Project

  • Github: https://github.com/tvoegerl/Meditative_Pomodoro

  • Website: https://tvoegerl.github.io/Meditative_Pomodoro/

Purpose

  • Pomodoro is a study technique where one focuses on studying for a given set of time followed by a small break. By continuing this cycle the brain can retain its attention span for more extended periods of study time. The goal of this project was to create a timer that keeps track of study time and then provides productive relaxing exercises to rejuvenate the brain during breaks.

Outcome

  • A website that allows for 4 combinations of study/break times.

  • A failsafe that tells when users should stop studying for the day.

  • Users having the option to choose which productive activity is best suited for them. Options include self-care, meditation, journal prompts, and exercises.

  • Programmers familiarized themselves with GitHub.

  • Allowed programmers to strengthen team-building skills.

  • Programmers developing JavaScript, HTML, and CSS skills.
