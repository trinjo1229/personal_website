Hydrogen Fuel Cell

Organization

  • Introduction to Engineering Course in Partner with BorgWarner

Project

  • Report: PDF Document

Purpose

  • The goal was to determine if creating a hydrogen fuel cell for a class 8 vehicle was possible, and if so should BorgWarner consider developing such technology.

Outcome

  • Determing that with the correct materials creating a hydrogen fuel cell for a class 8 vehicle was possible.

  • Creating a 3D reference model on CAD based on the calculated dimensions.

  • At the time, devloping such technology shouldn't be considered due to the difference in cost to maintain the fuel cell over time.
